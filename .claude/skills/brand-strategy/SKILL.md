# Brand Strategy — Name & Positioning for Ours App

Guide a structured brainstorming session for naming and positioning a family coordination AI app targeting the Israeli market.

## Product Context

Before generating names, internalize these facts — they shape what makes a good name:

- **What it is:** AI-powered family coordination app (tasks, shopping lists, calendar, rides, events)
- **Core interaction:** Natural language chat — you talk to it like a person
- **Target market:** Israeli families with school-age children
- **Language:** Hebrew-first, bilingual Hebrew/English
- **Positioning:** Structured layer ON TOP of WhatsApp, not a replacement ("Keep chatting on WhatsApp. Let [name] handle the action items.")
- **Competitors:** WhatsApp groups (default behavior, 99% penetration), Any.do (HIGH threat, Israeli-founded), Cozi, FamilyWall
- **Price:** 14.90 ILS/month (~$4)
- **Current brand:** Warm earthy palette (cream #F5F0E8, rust #C4714A, sage #4A7C59), serif wordmark (Cormorant Garamond)
- **Read `competitor-landscape-report.md`** for detailed competitive analysis before crafting positioning

## Session Flow

Run through these 3 phases in order. Present results after each phase and get user feedback before moving on.

### Phase 1: Name Generation

Generate **15-20 name candidates** organized by vibe:

**Vibe categories:**
- **Warm friend / helper** — feels like a trusted family member who keeps things running
- **Smart assistant** — capable, modern, efficient
- **Playful companion** — light, fun, makes coordination feel effortless
- **Hebrew-rooted** — draws from Hebrew sounds/roots but still works globally

**Name criteria (all must apply):**
1. **Human-like but invented/recognizable** — think Siri, Alexa, Gemini. Not necessarily a known Hebrew name. The name should *feel* like it could be a person, warm and approachable
2. **Bilingual phonetics** — pronounceable naturally in both Hebrew and English. No sounds that don't exist in one language (e.g., English "th" doesn't exist in Hebrew). Test: would an Israeli parent say this name comfortably in a Hebrew sentence?
3. **Female or unisex feel** — the app is a caring coordinator, not a command-line tool
4. **Short** — ideally 3-5 letters, max 6. Shorter = more memorable, easier to type, better as URL
5. **URL available** — check `.ai`, `.com`, `.app`, `.co` availability via web search. `.ai` is preferred for an AI product
6. **App Store clean** — search the App Store/Play Store to confirm no major clashing app exists with the same name
7. **No negative meanings** — check both Hebrew and English for unintended meanings, slang, or associations

**Output format per name:**

| Name | Hebrew | Phonetic | Vibe | Why it works | URL to check |
|------|--------|----------|------|-------------|--------------|
| Nuri | נורי | noo-REE | Warm friend | "My light" in Hebrew, gender-neutral, 4 letters | nuri.ai, nuri.app |

After presenting, ask: "Which names resonate? Any vibe you want more of? Any you want to drop?"

### Phase 2: Name Evaluation

Take the user's shortlist (or top 10 if no preference) and score each on a 1-5 scale:

| Criterion | Weight | What to evaluate |
|-----------|--------|-----------------|
| Bilingual flow | 25% | Does it roll off the tongue in both Hebrew and English? |
| Memorability | 20% | After hearing it once, would you remember it tomorrow? |
| Emotional fit | 20% | Does it evoke warmth, trust, family? Or cold/corporate/techy? |
| Domain available | 15% | Actually check via web search — is [name].ai or [name].app available? |
| App Store unique | 10% | No major competitor with same name in stores |
| Cultural safety | 10% | No negative slang, associations, or meanings in Hebrew/English/Arabic |

**Present as a ranked table** with total weighted score. Highlight top 5.

Then ask: "Ready to shortlist the top 3-5 for positioning work?"

### Phase 3: Positioning

For each finalist name, create a **brand positioning card**:

```
## [Name] — Brand Card

**One-liner:** [What the app is, in one sentence]

**Positioning statement:**
For [target audience] who [pain point],
[Name] is the [category] that [key benefit],
unlike [competitor] which [competitor weakness].

**Taglines (3 options each):**
English:
1. ...
2. ...
3. ...

Hebrew:
1. ...
2. ...
3. ...

**vs. WhatsApp:** [How to position — complement, not replace]
**vs. Any.do:** [Differentiation angle]
**vs. Cozi:** [Why this name wins]

**Name-product connection:** [Why THIS name fits THIS product]

**Risks:** [Any concerns with this name]
```

After presenting all cards, ask: "Which name + positioning feels right? Want to mix taglines across names?"

## Guidelines

- **Use web search** to check domain availability — don't guess. Check [name].ai, [name].com, [name].app
- **Use web search** to check App Store conflicts — search "[name] app" on Google
- **Hebrew phonetics matter deeply** — Israeli word-of-mouth is the #1 growth channel. If a name is awkward to say in Hebrew conversation, it will fail regardless of how good the brand strategy is
- **The name will be spoken aloud constantly** — "תשאלי את [name] מה יש היום" ("Ask [name] what's on today"). Test every candidate in a Hebrew sentence like this
- **Don't over-explain** — present names and let the user react. The best name will feel obvious when they see it
- **Read `competitor-landscape-report.md`** before Phase 3 — the positioning must be grounded in real competitive dynamics, not generic marketing speak
